# Para a Mariana 💛 — guia rápido

Um site de Dia dos Namorados pra abrir no celular. Tudo num arquivo só (`index.html`) + uma pasta de fotos.

## 📁 O que tem aqui
- `index.html` — o site inteiro (abra no navegador pra ver).
- `fotos/` — as fotos do mural (`1.jpg`, `2.jpg` … `6.jpg`). Já vêm com imagens "placeholder"; é só substituir pelas suas.

## 🚀 Como publicar no GitHub Pages
1. Crie um repositório novo no GitHub (ex.: `para-a-mariana`).
2. Envie o `index.html` e a pasta `fotos/` pra dentro dele (botão **Add file → Upload files**).
3. No repositório, vá em **Settings → Pages**.
4. Em **Branch**, escolha `main` e a pasta `/ (root)` e clique **Save**.
5. Em ~1 minuto o site fica no ar num endereço tipo
   `https://SEU-USUARIO.github.io/para-a-mariana/` — abra no celular dela. 🎉

## ✏️ Como trocar os textos
Abra o `index.html` em qualquer editor de texto (até o Bloco de Notas serve), procure por **`EDITE AQUI`** lá no começo. Tudo que dá pra trocar está dentro de aspas, no bloco `CONFIG`:
- nomes, data do namoro, título da carta e os parágrafos;
- linha do tempo (datas e momentos de vocês);
- legendas das fotos;
- música principal + playlist (e dá pra colar um link do Spotify/YouTube em `musicaLink`);
- lugares que já foram + sonhos de viagem;
- os vales de amor.

## 🖼️ Como trocar as fotos
Coloque suas fotos na pasta `fotos/` com os nomes `1.jpg`, `2.jpg`, …, `6.jpg`
(substituindo as que já estão lá). Pronto — elas aparecem no mural automaticamente.
Dica: fotos mais ou menos quadradas ficam melhores.

## 💡 Detalhes fofos
- O contador de dias se atualiza sozinho a partir da data em `dataInicio`.
- Os "vales de amor" ficam marcados como **resgatados** no celular dela quando ela toca neles.
- Toque numa foto do mural pra vê-la maior.
